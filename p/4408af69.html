什么是问题？

Posted on 2019-06-12 Edited on 2021-11-26 Views: Valine:

最近接触到一个概念，我觉得影响很大，记录一下。

创造力的起点，是正确表述问题，是看见矛盾，承认这个矛盾，然后到别处找到方案，解决它的过程。创新的本质是找到万事万务内含的那个矛盾，然后把它转化成别的矛盾。

本文价值：问题的本质

所以洞察力就是一眼能抓住事物的核心矛盾的人，有时并不是消灭它，而是转化成其它的矛盾，将尖锐棘手的替换成其他的问题，用时间维度去再再层层替换按可操作的空间，你可以说它是苟延残喘、治标不治本，但现实很多问题就是这样，很难达到理想的状态，哪怕国家这种超级智囊团对房价的控制。

怎么样是提出好的问题

英语中对于问题有不同的词汇表达，分别是Question,,Problem,Puzzle。通过辨析，其实可以有些收获。

QUESTION

Question 是需要回答的问题是疑问，往往表述模糊的问题。
如：我怎么才能幸福？人生的道路该怎么走？
很多人只能问出这样的问题，但其实他并不知道他到底在问什么。可以得到别人的回答，但得不到自己的答案。

PROBLEM

Problem 是需要解决的问题，这个词指的是明确界定下来的问题。
如：公司今年利润下滑，怎么办？体重超标，怎么办？
这种问题就比较清淅，可以解答了，能得到解决方案。
If there is a question, you need an answer. If there is a problem, you need a solution.
但是这还不是最好的问题，因为答案会有很多种，问问题的人还是搞不清该怎么选择？
Puzzle才是最好的问题。

PUZZLE

Puzzle 是非常清晰的，是被良好定义过的问题。
如：我们中小学做的数学题。
Puzzle 通常有明确的解决方向，像解谜一样解开。

一起举个例子：
Question: 如何设计一款太空望远镜？
Problem: 如何解决太空望远镜的口径大小和运输可先进性之间的矛盾？
Puzzle:如何借鉴折纸的方法，设计一款太空望远镜的可折叠结构？
最终通过puzzle的提问方式，像数学题一样解开它。

很多科学问题，都是puzzle。

当我们遇见问题时，不妨试试提高问题的深度，多提“puzzle”。

问题的分类

如果将问题进行分类，可以是单纯的问题，两难的问题和棘手的问题，通过理解问题也许更能明确问题。

单纯的问题

有明确的方向，有能让人放心的答案，解决了就可以宣布胜利。完全能够胜任。

两难的问题

没有唯一正确的方向，通常会产生纠结。比如迷茫。
大学生迷茫的本质就是问题类型切换了，之前都是有固定的方向备考升学，现在切换成了两难问题，没有目标没有经验没有道路，所以就迷。

棘手的问题

走入社会，承担责任时，要做决定或替别人做主时，就会出现棘手问题。
棘手问题，不是用来解决的，也许跟本解决不了，只能应付，做好跟它长期共处的准备。问一个好的puzzle,创新式的转化矛盾。

三种困难的区别是，就能理解在复杂的局面里，关键局部的关键人物或指标，才决定了全局的成败。
