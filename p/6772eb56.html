获取音乐相关链接文件

Posted on 2021-10-04 In Life Views: Valine:

背景：

原来使用刘志进实验室可以很方便的获取相关内容，现在不能用了。
就还挺可惜的。
为了定期能让自己的网站可以播放希望的音乐，着手去解决这个问题。

本文价值：获取音乐相关链接

需求：

获得网易云音乐歌词文本，音乐直链接，歌曲封面图片应用于网站上播放使用。

需求实现

1. 网易云音乐歌词文本

通过小工具获取

下载小插件：网易云歌词下载

链接：https://pan.baidu.com/s/176Ws-DN43IxXl-ZiR5wAnw
提取码：jres
以这首歌为例：获得了分享链接如下
http://music.163.com/song?id=549941712&userid=58611551
搜索选择UTF-8编码，格式 歌手名-歌名

2. 音乐直链接

分享链接中有song id 和 user id ，我们选择复制song id
直链接如下：
http://music.163.com/song/media/outer/url?id=549941712.mp3
浏览器打开显示是这样的，会自动播放音频

3. 歌曲封面图

还是用获得的分享链接在浏览器打开，得到如下

通过搜索源代码中图片尺寸（130x130）可快速获得封面图链接。
打开F12 => Ctrl F => 130

点开框中的链接就能得到封面链接
https://p1.music.126.net/v1Uk3MbeZ2dlF-Og9FclEQ==/109951163237368778.jpg?param=130y130

结束

最终有了上面这些链接，我就能放到网站上播放了
